(static) getCoursesByName(search_term, account_id, search_by, blueprint, enrollment_term_id) → {Array.<object>}

Get courses that match the criteria.
Parameters:
Name Type Description
search_term string The partial course name, code, or full ID to match and return in the results list. Must be at least 3 characters. e.g., GEDE-2300
account_id number account id e.g., 1
search_by string The filter to search by. course|teacher. “course” searches for course names, course codes, and SIS IDs. “teacher” searches for teacher names e.g., course
blueprint boolean If true, include only blueprint courses. If false, exclude them. If not present, do not filter on this basis. e.g., true
enrollment_term_id number If set, only includes courses from the specified term. e.g., 1
