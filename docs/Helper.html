(static) fillValues(startrow, startcol, jsonobject, columns, bgcolor)

Fill a SpreadSheet range with a json object. If the jsonobject is an array, it will be displayed in rows. If the jsonobject is an object, it will be display vertically.
Parameters:
Name Type Description
startrow number An absolute row index of a sheet
startcol number An absolute column index of a sheet
jsonobject Object data to be displayed
columns string A DISPLAYCOLUMNS name defined in the DISPLAYCOLUMNS variable. Check DISPLAYCOLUMNS.js. If provided, only the fields defined in the DISPLAYCOLUMNS will be displayed. If not provided, all original field of the jsonobject will be displayed.
bgcolor string Provide a RGB color for the background of the filled area . If not provided, a randome color will be used. See function getColor() for all colors
